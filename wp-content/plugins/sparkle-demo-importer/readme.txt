== Description ==

Sparkle Demo Importer imports sparkle themes full demo with just one click. It is specially developed for demo import purpose. This plugin works for theme developed by SparleThemes and if other themes wants to use then they have to use **action** filter to work.

You just need to define the array that includes the location of the demo zip files and other related info.

The demo zip should contain the XML file, customizer (.dat) file, widget (.wie) file. It is not necessary to add all these files in the demo zip. You can skip the files if your demo does not need it.

= Features =
* Import Complete Demo Data
* Automatic Install Required Plugins
* Import Widgets Data
* Import Customizer Data
* Reset Site
